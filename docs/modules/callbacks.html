Neorg User Callbacks File

User callbacks are ways for the user to directly interact with Neorg and respond on certain events.

Functions

neorg.callbacks.on_event (event_name, callback, content_filter)
Registers a new callback Triggers a new callback to execute whenever an event of the requested type is executed

Parameters:

  • event_name string The full path to the event we want to listen on
  • callback function The function to call whenever our event gets triggered
  • content_filter function A filtering function to test if a certain event meets our expectations
neorg.callbacks.handle_callbacks (event)
Handles the triggering of all registered callbacks Used internally by Neorg to call all callbacks with an event

Parameters:

  • event table An event as returned by neorg.events.create()
